eosio::name  status

Метод установки статуса

AUTH: _me

Метод устанавливает специальный статус лидера (headman) для партнёра. Обладая статусом headman, все свободные регистрации новых партнёров (без указания referer) устанавливаются под лидеров в случайном порядке.

Parameters
[in]usernameимя аккаунта пользователя для обновления статуса
[in]statusимя статуса партнёра (сейчас доступен только headman)
